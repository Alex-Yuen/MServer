# Glossary

* client - 客户端
* server - 服务器
* packet - 网络包
* player - 玩家对象
* clt    - client简写
* srv    - server简写
* cmd    - command简写，网络指令
* conn   - connection简写，网络连接
* srv_conn - 服务器之间网络连接
* clt_conn - 客户端与服务器之前网络连接
* stream - 二进制流
* cfg    - config简写，配置
* android - 机器人
* guild   - 公会(帮派)
* scene   - 场景
* dungeon - 副本(或者直接叫fuben)
